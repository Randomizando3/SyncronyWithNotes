# SyncronyWithNotes

Aplicativo em .NET MAUI com proposta de ritmo musical, sincronizacao de notas e interacao do usuario com faixas e velocidades diferentes.

## Recursos identificados

- Selecao de musica
- Controle de velocidade
- Notas caindo em pistas
- Zona de acerto para interacao
- Persistencia local de musicas e notas

## Estrutura principal

- `MainPage.xaml`: interface principal do jogo
- `Models`: entidades de musica e notas
- `Data/AppDatabase.cs`: persistencia local
- `Services`: audio, seed de musicas e servicos auxiliares

## Como executar

1. Abra `SyncronyWithNotes/SyncronyWithNotes.sln` no Visual Studio 2022.
2. Garanta a instalacao do workload do .NET MAUI.
3. Restaure os pacotes e rode na plataforma desejada.
