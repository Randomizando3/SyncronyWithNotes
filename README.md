# SyncronyWithNotes

## English
SyncronyWithNotes is a .NET MAUI app centered on rhythm interaction, music selection, falling notes, and local data used to support the gameplay flow.

### Features identified
- Song selection
- Speed control
- Falling note lanes
- Hit zone interaction
- Local persistence for songs and notes

### Project structure
- `MainPage.xaml`: main gameplay interface
- `Models`: song and note entities
- `Data/AppDatabase.cs`: local persistence layer
- `Services`: audio, seeding, and support services

### Run locally
1. Open `SyncronyWithNotes/SyncronyWithNotes.sln` in Visual Studio 2022.
2. Make sure the .NET MAUI workload is installed.
3. Restore packages and run on the selected platform.
